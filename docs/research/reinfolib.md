# 不動産情報ライブラリ API 仕様・規約調査（2026-07-09・Actor#3 Step 0）

一次情報: reinfolib.mlit.go.jp の公式APIマニュアル（XIT001/XIT002）・利用規約（サイト規約＋API利用規約全16条）・「コンテンツの出典」。実挙動は2026-07-09に実キーで検証。

## XIT001（不動産価格情報取得API）【マニュアル＋実証】

- 認証: ヘッダ `Ocp-Apim-Subscription-Key`。応答はgzip JSON（Node fetch/undiciは透過伸長）
- パラメータ: `priceClassification`（**01=不動産取引価格情報のみ／02=成約価格情報のみ／未指定=両方**＝要件書§12未決#4解消。実証: 千代田区2024で549=278+271件）／`year`必須（取引価格2005年〜・成約価格2021年〜）／`quarter`（マニュアル表記は必須だが**実証では省略可＝通年返却**。Actorは任意扱い）／`area`2桁・`city`5桁・`station`6桁（**少なくとも1つ必須**。stationは国土数値情報N02_005gの駅グループコード）／`language`（ja/en、未指定=ja）
- レスポンス: 全29フィールド・**フィールド名は英語固定・全値string（該当なしは空文字）**。languageは※2印フィールドの**値のみ翻訳**（TradePrice/Area等の数値系は非翻訳）。ja/enで**件数・順序が一致**（実証: TradePrice列一致）→**同一クエリのja/en二重取得をindexで結合**し、TradePrice/MunicipalityCodeの一致でサニティ確認する設計（判断はdecisions.mdへ）
- `BuildingYear`は西暦「YYYY年」（ja）/「YYYY」（en）、空文字あり（実証）。和暦は観測されず
- **404=該当なし・0件**（エラーにしない）。データなし年で実証済み

## XIT002（市区町村一覧）【実証】

`area`必須2桁＋`language`（**en対応: "Chiyoda Ward"等**）→ 名称→5桁コード解決に使用。

## 利用規約【原文引用は調査ログ参照】

- **PDL1.0**（公共データ利用規約・CC BY 4.0互換）適用。取引価格・成約価格とも適用除外リスト外→**出典記載を条件に商用二次利用・第三者提供可**と解釈。API規約第7条（クレジット表示）・第8条（API利用者が開発・運営責任を負う旨のエンドユーザ明示）は第三者向けサービス提供を前提とした条文→**オペレータのAPIキーで第三者にデータ提供する形態は可**と判断し、**BYOキー入力欄は設けない**
- **第9条（譲渡禁止）**: 利用契約上の地位・権利義務の第三者移転は不可→**APIキー自体の貸与・共有・配布はしない**（サーバ側でのみ使用）
- レート: 「明確な上限はないが間隔を空けて」→**直列1req/秒**を維持（N-1）
- **クレジット指定文言（第7条・全サービス共通）**: 「このサービスは、国土交通省の不動産情報ライブラリのAPI機能を使用していますが、提供情報の最新性、正確性、完全性等が保証されたものではありません」＝**attributionパッケージの既存文言（引継書§4.6）と一致・これで確定**
- 出典記載例: 「出典：国土交通省　不動産情報ライブラリ（URL）」＋**編集・加工して利用する場合は加工した旨の記載義務**（「あたかも国が作成したかのような態様」の禁止）→READMEのattribution節に反映
- **成約価格情報に固有の追加クレジットは規約上見つからず**（定義: レインズ保有情報を国交省が加工しRMIで公表するもの）。RMI（reins.or.jp）側規約は未確認→READMEで成約価格の由来を明示し、要確認事項として残す
- 第15条: 予告なき提供停止・変更あり（賠償なし）→README免責。第5条: キーの不正利用疑い・長期未アクセス等で利用停止あり
